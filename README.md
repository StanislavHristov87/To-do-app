# 📝 To-Do App (React)

A simple and modern To-Do application built with React.
This project is part of my learning journey as a junior frontend developer.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Save tasks (localStorage / Firebase)
* ⚡ Fast and responsive UI

## 🛠️ Technologies Used

* React (useState, useEffect)
* JavaScript (ES6+)
